# DSS3D — Портфолио разработчика

Одностраничное портфолио/визитка: секции «Обо мне», «Проекты», «Контакты», фильтрация проектов, модальное окно с деталями и форма связи (mailto).

## Стек

- React + TypeScript
- Vite
- Tailwind CSS

## Запуск локально

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
npm run preview
```

## Как изменить контент

Все данные вынесены в файл:

- `src/content/site.ts`

Там можно поменять имя, роль, навыки, проекты и контакты.

## SEO

Базовые метатеги и JSON-LD находятся в `index.html`.
